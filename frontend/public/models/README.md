# 3D models for the game

Place your **OBJ** or **GLB/GLTF** file here.

**Trees (replaces default pink trees):**
- Name it **`tree.glb`** or **`tree.obj`** and it will be used automatically.
- Or set in `.env.local`: `NEXT_PUBLIC_TREE_MODEL_URL=/models/yourfile.glb`

**Checkpoints (optional):**  
Set `NEXT_PUBLIC_CHECKPOINT_MODEL_URL=/models/checkpoint.glb` in `.env.local`.

Supported: **.glb**, **.gltf**, **.obj**. Models are auto-scaled to fit the scene.
